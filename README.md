# Temp-Mail
- INFINITY NETWORK Temp-Mail
- python
- php
- version V1.0
## Installing
```ruby
apt-get update
apt-get upgrade
apt-get install git
apt-get install python
apt-get install python2
apt-get install php
pip install requests
pip install omero-marshal
git clone https://github.com/infinitynetwork13/Temp-Mail
```
## Tool Run
```bash
cd Temp-Mail
python main.py
```
